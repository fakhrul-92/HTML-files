Today's topic is- Attributes in the pictures

This lesson contains
  1. Changing border color for a picture: Use style=.. ".. border-color: color name" inside your img tag. You can use RGB with hastag instead
  2. Leaving space in between pictures: Use & n b s p to leave space
  3. Partial loading of pictures.: Use alt=.. ".. some text.." inside image tag
  4. Inside text alt=.. ".. some text.." is visible too if the picture is missing. See this effect on 24th lecture (the same file as this one)

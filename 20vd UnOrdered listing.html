Today's topic is- creating UnOrdered list

Ordered list and UnOrdered list are similar in format. UnOrdered list is called bullet list too.
UnOrdered listing takes Type=" " attribute too. It has has 3-types of bullets. They're-

  • circle
  • squre
  • disc
The important application is, we use UnOrdered list to create Menue Tabs for a webpage!

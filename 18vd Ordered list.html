Today's topic is- creating list in a webpage

From now on, in the footer section, we will be using listing. When we use listing tags, we need 2 types of listing tags.

  1. listing object separator. It is li only
  2. order declation tag. It is mainly ordered, unordered & definition list.

There are 3types of lists in a webpage. They are-
          - Ordered list: Tag is- ol
                    Format: OL-li.. .. .. .. (list objects)-li-OL
        - Un-ordered list: Tag is- ul
                    Format:
        - Definition list: Tag is- dl

By using them, we can scape from blockquote tag. We will be sorting our list.
Note:
  • In ordered list, things are normally listed in order with 1, 2, 3 etc
  • In Unordered list, things are normally listed in order with BIG_DOT
  • If you intend to use a, b, c, .. .. Use type="" attribute and inside "" type a. Then complete your listing
  • In type attribute, you can use A-for-capital-listing, i-for-roman-listing
  • To start from numbers grater than 1 in the list, like, 8, 9, 10, .. .. use start="" attribute . Inside "" type 8 to start listing from 8

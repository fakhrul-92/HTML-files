Moving image (Advance in marquee tag )

we will be using tags below

  • marquee| for moving
  • img| for embedding pictures
  • src..=.."..photoName with format.."| attribute to import the photo name
  • behavior..=.."..slide/alternate.." | attribute to move like come-&-go
  • hspace..=.."..no.." | = HORIZONTAL SPACE. It is used to leave spaces from both sides
  • vspace..=.."..no.." | = VERTICAL SPACE. It is used to leave spaces from TOP+BOTTOM both sides
  • Loop..=.."..no.." | write 1 for 1time repeat, 2 for 2times repeat, 3 for 3times...
  • scrollamount..=.."..no.." | you can increase movement pace by using this attribute
  • height..=.."..No+px.." | to determine how heigh the moving will be
  • width..=.."..No+px.." | to determine how wide the moving will be
  • style..=..".." | style attributes are applicable

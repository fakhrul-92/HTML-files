More table formatting

More table attributes
1. table-head: thead |
2. table-body: tbody |
3. table-foot: tfoot |


Some observations:
     1. First, prepare the layout for the table. Set the border to see the skeleton of the table. Without border, table is not visible.
     2. Choose special colum and declare the width of special colums
     3. Write your caption for the table
     4. Introduce height and width of the table. h-20, w-100 could be the standard. Change and observe the result
     5. Fill out your heading
     6. Merging: Remove some td tags , Inside td tag define the number for merging, with column-span colspan tag
     7. Input all your data
     8. Choose tr-table row and declare the height of the header and footer row

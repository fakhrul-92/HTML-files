More attributes of a TABLE for a webpage

More table attributes
1. Height: height..=.."..number.."
2. Width: height..=.."..number.."
3. align: align..=.."..left (default)/ center/ right.."
4. valign: valign..=.."..baseline/ top/ middle/ bottom.."
5. colspan: colspan..=.."..number.." | It works as merge , write 2 to merge 2 cells
6. style> font-size: font-size..:..Number with px
7. style> Color (font color): color..:..colorName/ colorCode
8. style> background-color: background-color:ColorName/ colorCode
9. background: (image): background="image.format"
10. cellspacing: cellspacing=Number with px


Some observations:
     1. First, prepare the layout for the table. Set the border to see the skeleton of the table. Without border, table is not visible.
     2. Write your caption for the table
     3. Introduce height and width of the table. h-20, w-100 could be the standard. Change and observe the result
     4. Fill out your heading
     5. Merging: Remove some td tags , Inside td tag define the number for merging, with column-span colspan tag
     6. Input all your data

Today's topic is- Changing background color of a webpage & colorCode

To change the background color-
                    - Go to the body tag on top of the page.
                    - inject | style = "".
                    - inside double quote specify the color name in background-color:ColorName .
                    - Instead of color name , you can introduce color code too.

Use of R G B in color code-
                    - use a hash # instead of color name.
                    - Reserve 6-blank spaces for color in percentage . The format becomes # ab cd ef
                    - Use respectively two spaces ab for red color, ab for green color, ab for blue color.

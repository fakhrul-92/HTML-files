Crating Styling a TABLE for a webpage

Before you start
-We have 3table tags:
  1. th- table head | Describes names of columns
  2. td- table data | Points to a cell & its data
  3. tr- table row | Refers to a certain whole row of the table
-Tag-1: style:
      text-align: center/ left (by default), right. border-color:color name
      border-color: color name. You can use 4 different color clockwise
      color: color name for your text.
      border-style: style for border can be- dottet/ dashed/ solid/ double/ none/ hidden/ groove/ ridge/ inset/outset/ initial
      background-color: set the backgroud color or color code
      background: set the backgroud image for the table
Use all these attributes inside the double qoute of style and remembe, separator is semicolon (;) this style attribute belongs to the table tag. YOU CAN SET BG-IMAGE FOR DEDICATED CELL OR ROW!!
